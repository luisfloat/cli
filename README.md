# luisfloat-cli

It's my home CLI UI.

## Technologies

* **JS(X)** for UI scripts;
* **React** as library for building UI;
* **Ink** as CLI component library for React.

## Usage

Requirements: `node >= 10`

```bash
npx luisfloat/cli
```

Type `q` to leave.
